# Equipment

This section contains documentation for instruments, machines and self-developed laboratory systems.

## Equipment categories

- Lithography systems
- Spin coaters
- Hotplates and furnaces
- Microscopes
- Probe stations
- Measurement equipment
- Wet-chemical equipment
- Open-source laboratory equipment

Each equipment page should contain:

- `purpose`;
- `location`;
- `responsible person`;
- `required training`;
- `basic operating procedure`;
- `maintenance information`;
- `known problems`;
- `related processes and safety documents`.
